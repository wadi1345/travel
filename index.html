🌸 仙台六日遊

點擊下方有虛線的地標，地圖會自動飛過去喔！

🛂 重要證件與票券
  • 護照： 效期需在6個月以上。
  • 數位入境碼： 提前填寫好 Visit Japan Web 並截圖。
  • 機票與住宿： 星宇航空電子機票、Airbnb 入住資訊。
💰 財務與支付
  • 日幣現金： 仙台朝市、大崎八幡宮、部分老街小吃必備。
  • 信用卡： 帶 2-3 張高海外回饋卡片。
  • 交通卡： 實體 Suica/Pasmo 或綁定 Apple Pay。
📱 電子設備
  • 上網工具： eSIM 卡或 Wi-Fi 機。
  • 行動電源： 拍照導航極耗電，必帶。
💊 隨身藥品與生活
  • 常備藥品： 腸胃藥 (吃海鮮必備)、感冒藥、止痛藥。
  • 衣物： 5月中溫差大，採「洋蔥式穿法」，帶輕薄防風外套。
  • 好走鞋子： 景點階梯多，平底鞋非常重要。
  • 折疊購物袋： 西友超市和 OUTLET 採買必備。
🍣 鹽釜在地海鮮與壽司
  • すし哲本店：殿堂級名店，大目鮪魚極度推薦。
  • 鮨しらはた：緊鄰すし哲，食材處理細膩。
  • 廻鮮寿司塩釜港：東北最強迴轉壽司(Marine Gate店)。
  • 亀喜寿司：老字號頂級壽司備案。
🥩 燒肉、居酒屋與義式
  • 炭火焼肉 まさにく：高CP值黑毛和牛與厚切牛舌。
  • まるきた：隱藏版居酒屋，生魚片份量大。
  • BLAIR MARINA：結合在地海鮮的義式料理與港口夜景。
🍢 仙台與松島特色
  • 伊達の牛たん本舗：必點極厚芯牛舌。
  • 松島魚市場：產地直送海鮮丼與現烤牡蠣。
  • Pensee (パンセ)：老街必吃牡蠣咖哩麵包。
  • Kirin Beer Port 仙台：啤酒廠附設的烤羊肉。
🍰 經典名菓與甜點
  • 萩之月 (菓匠三全)：鬆軟蜂蜜蛋糕包裹卡士達醬。
  • 喜久福 / ずんだ茶寮：各種口味大福與毛豆奶昔。
  • 白松最中：傳統日式和菓子 (仙台站內有售)。
🍘 鹹食與限定零嘴
  • 牛舌製品 / KitKat / Saichi：牛舌餅與仙台限定零嘴全制霸。
  • 松島蒲鉾本舗：仙台在地經典竹葉魚板。
  • 浦霞 酒Gallery：鹽釜在地酒造限定酒款。
